A drop-in replacement for [`Button`](https://mui.com/material-ui/react-button/) that adds a `pill` shape modifier. All props from the underlying MUI `Button` (variant, color, size, startIcon, endIcon, href, …) are forwarded.

## Import

```tsx
import { MoarButton } from 'moar-mui';
// or — same code, smallest import graph:
import { MoarButton } from 'moar-mui/MoarButton';
```

## Theming

`MoarButton` uses Emotion's `styled` factory and reads from the active MUI theme, so any `components.MuiButton` overrides on your `ThemeProvider` apply automatically:

```tsx
import { createMoarTheme } from 'moar-mui';

const theme = createMoarTheme({
  components: {
    MuiButton: {
      styleOverrides: {
        root: { textTransform: 'none' },
      },
    },
  },
});
```
